# Ведение проекта

Эта папка хранит рабочую структуру проекта: генеральное ТЗ, SEO-архитектуру, контентную модель, граф связей, города, услуги, решения и чеклисты.

Основные документы:

- `000-project-core.md` — главный документ проекта.
- `010-seo-architecture.md` — SEO-структура, URL, canonical, sitemap.
- `020-content-model.md` — сущности, поля, Content Collections.
- `030-link-graph.md` — primary, secondary, relations, перелинковка.
- `040-localization.md` — города, городские хабы, локальные блоки.
- `050-calculator.md` — логика калькулятора.
- `060-copywriting-rules.md` — правила текстов, FAQ, мета, структура страниц.
- `070-astro-implementation.md` — правила реализации в Astro SSG.
- `080-qa-checklist.md` — чеклист проверки.

Папки:

- `briefs/` — ТЗ на отдельные страницы.
- `decisions/` — зафиксированные архитектурные решения.
- `data-drafts/` — черновики JSON/MD-моделей данных.
